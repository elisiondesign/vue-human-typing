# vue-human-typing
> Vue component to create human-like text typing effect.

## Usage
```html
<human-typing :text="project.categories" :options="{...}"></human-typing>
```
```json
options : {
    delay: Number, 
    speed: Number, 
    speedDeviation: Number
}
```
